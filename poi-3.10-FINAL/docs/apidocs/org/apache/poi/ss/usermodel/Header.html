Interface Header

  • All Superinterfaces:
    HeaderFooter
    All Known Implementing Classes:
    HSSFHeader, XSSFEvenHeader, XSSFFirstHeader, XSSFOddHeader

    public interface Header
extends HeaderFooter
    Common definition of a HSSF or XSSF page header. For a list of all the different fields that can be placed into a header, such as page number, bold, underline etc, see HeaderFooter.
